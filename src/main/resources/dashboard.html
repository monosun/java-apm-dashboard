모니터링 대상 미설정

monitor.properties에 JMX 대상을 설정하면 Tomcat / Spring Boot 등의 JVM을 여기서 모니터링합니다.

remote.jmx.enabled=true   remote.jmx.host=<HOST>   remote.jmx.port=9999

java -javaagent:java-monitor-1.6.0-agent.jar=port=7979 -jar your-app.jar
